# Inheritance Example

A simple Java application demonstrating the Object-Oriented Programming (OOP) concept of Inheritance.

## Features

- Parent class (Person)
- Child class (Employee)
- Constructor inheritance using `super()`
- Display inherited and child class data

## Technologies Used

- Java
- IntelliJ IDEA
- Git
- GitHub

## Concepts Covered

- Inheritance
- extends
- super()
- Constructors
- Methods
